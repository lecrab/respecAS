Beheer Afsprakenstelsel eToegang : DeprecatedActingSubjectID (XACML)

Created by Sander Boer, last modified on May 28, 2019

Description

A XACML Attribute element containing the specific pseudonym of the user, for purposes of migrating user identifiers.The element MUST NOT contain other attributes (@) than those described here.

@AttributeId

urn:etoegang:core:DeprecatedActingSubjectID

@Datatype

http://www.w3.org/2001/XMLSchema#string

@IssuerEntityID of the issuing MR

AttributeValue

The NameQualifier of the NameID after decryption holds the used type, see Specific pseudonym.
