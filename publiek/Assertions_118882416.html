Beheer Afsprakenstelsel eToegang : Assertions

Created by Sander Boer, last modified on Mar 05, 2018

 

 

Description

 A XACML element that contains an MR <authorization assertion> for checking authorization in GUC9 Token refresh (native apps).

@AttributeId

urn:etoegang:core:Assertions

@Datatype

urn:oasis:names:tc:SAML:2.0:assertion#Assertion

@Issuer

EntityID of the Herkenningsmakelaar

AttributeValue

copy of  SAML <assertion> or XACML <authorization assertion> in XML form, copied such that Signature still can be validated.
